### 📌 기능
- 서울시 내 모든 유치원 위생 환경 조회
- 해당 유치원의 지도, 상세정보 조회
- 원하는 유치원을 따로 저장할 수 있다.

### 📌 개발 환경
- iOS 16.0 +
- Xcode 14.3

### 📌 사용 기술 및 라이브러리
- UIKit
- CocoaPods
- MVC Pattern
- Google Maps API
